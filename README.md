# Website Bán Thực Phẩm (Ruby on Rails)

## Giới thiệu
Website bán thực phẩm trực tuyến, xây dựng bằng Ruby on Rails.  
Cho phép hiển thị sản phẩm, quản lý giỏ hàng, đặt hàng và quản lý sản phẩm cho admin.

## Công nghệ sử dụng
- Ruby 3.3.5 (2024-09-03 revision ef084cc8f4)
- Rails 7.2.2
- SQLite 3
- HTML, CSS, JavaScript
- Bootstrap / Tailwind CSS

## Chức năng chính
- Quản lý sản phẩm: Thêm, sửa, xóa sản phẩm. Quản lý thông tin: tên, mô tả, giá, hình ảnh, loại sản phẩm.
- Quản lý đơn hàng: Theo dõi từ lúc đặt đến khi giao hàng. Lưu thông tin: số lượng, ngày đặt, trạng thái (đang xử lý, đã giao).
- Quản lý khách hàng: Lưu trữ thông tin: họ tên, địa chỉ, email, số điện thoại, hỗ trợ chăm sóc và xử lý đơn hàng.
- Chức năng thanh toán: Hỗ trợ thanh toán online qua PayPal, Stripe, MoMo, v.v.
- Giao diện người dùng: Thân thiện, dễ sử dụng cho cả khách hàng và chủ shop.
- Tìm kiếm và lọc sản phẩm: Hỗ trợ tìm kiếm theo tên và loại sản phẩm.
- Thống kê các đơn hàng: Theo dõi doanh thu, doanh số đơn hàng trong khoảng thời gian.
- Bảo mật: Bảo vệ thông tin khách hàng và đơn hàng.
